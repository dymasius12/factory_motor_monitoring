Dashboard & Demo

Real-time Monitoring Dashboard

The React-based dashboard provides maintenance teams with real-time visibility into motor health status and alerts.

Key Features:

1. Active Alerts Panel

Real-time display of current alerts with sensor readings and timestamps

2. Motor Status Cards

At-a-glance health status for each motor (Normal, Warning, Critical)

3. Daily Alert Charts

Bar charts showing alert frequency by motor over recent days

4. Alert Trend Analysis

Line chart showing total daily alerts across all motors

Technical Implementation:

  • React with Tailwind CSS for UI
  • Recharts for data visualization
  • WebSocket/polling for real-time updates
  • Responsive design for multiple devices
