Node.js Data Ingestion Service

Sensor Data Ingestion

The Node.js service provides an HTTP API endpoint for receiving sensor data, validating it, and publishing alerts when thresholds are exceeded.

Key Features:

API Endpoint

POST /api/sensor for JSON payload

Input Validation

Ensures valid field values

Threshold Checking

Detects abnormal readings

Alert Publishing

Sends to RabbitMQ exchange

Sample Request:

// POST /api/sensor
{
  "motorId": "MTR-01",
  "timestamp": "2025-07-17T10:15:00Z",
  "vibration": 3.2,
  "temperature": 85.1
}
// Threshold checking function
function checkThresholds(data) {
  const { vibration, temperature } = data;
  const alerts = [];
  
  // Vibration threshold: > 2.5 g
  if (vibration > 2.5) {
    alerts.push({
      motorId: data.motorId,
      sensorType: 'vibration',
      value: vibration
    });
  }
  
  return alerts;
}
